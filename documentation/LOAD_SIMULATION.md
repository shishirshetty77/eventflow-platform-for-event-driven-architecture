## âš™ï¸ Configuration

You can adjust the simulation behavior through environment variables:

| Variable | Default | Description |
|----------|---------|-------------|
| `METRICS_INTERVAL` | `5s` | How often to generate metrics |
| `LOGS_INTERVAL` | `3s` | How often to generate logs |

To modify these, update the service configuration in `docker-compose.yml`:

```yaml
services:
  auth:
    environment:
      - METRICS_INTERVAL=10s   # Slower metric generation
      - LOGS_INTERVAL=5s       # Slower log generation
```
